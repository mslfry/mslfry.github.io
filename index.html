How to create a webpage on Github:

On your MPHS Laptop:

  1. open file explorer
  2. create a folder for your webpage files
  3. open the Notepad application and create a New File
  4. use your new HTML/CSS and JavaScript knowledge to build your page!
  5. this file MUST be named EXACTLY "index.html" for the Github web server to recognize it as your main webpage

On github.com:

  • Create an account using your school username and email address
  • Verify your email address in your school email account
  • On the start page, click 'Start a project'
  • -->You MUST name your project your_username.github.io in order for Github to recognize your project as a webpage!
  • -->Select the option to 'Initialize this repository with a README'
  • Return to your home page and select your webpage name under Repositories
  • Upload your files by using the Upload Files option.(you will need to scroll down to the bottom of the page to commit your changes)
  • It can take a few minutes for the Github web server to see your new files

On Canvas:

  • See Webpage Project assignment in the Final Project module for grading requirements and rubric
